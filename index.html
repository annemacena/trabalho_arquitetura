* Antes de mais nada, esse projeto foi desenvolvido para conclusão da disciplina de Arquitetura de Computadores I da Universidade Federal de Sergipe - UFS.

O projeto tenta simular em software uma CPU simples, mostrando o fluxo de dados após a leitura de uma palavra de instrução na memória. Temos:

  • Palavra de memória de 16 bits.
  • Duas operações aritméticas (ADD, SUB),
  • Duas operações lógicas bit a bit (OR, AND),
  • Uma instrução de movimento de dados (MOV),
  • Uma instrução de salto incondicional (JMP).
